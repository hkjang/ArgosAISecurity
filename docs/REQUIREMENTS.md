# Argos AI Security 개발 요건서

## 1. 제품 개요

| 항목    | 내용                                                        |
| ----- | --------------------------------------------------------- |
| 제품명   | Argos AI Security                                         |
| 제품 유형 | AI 기반 Linux 서버 보안 플랫폼                                     |
| 핵심 목표 | Linux 서버의 랜섬웨어, 이상 행위, 권한 상승, 파일 변조, 내부자 위협을 실시간 탐지·차단·복구 |
| 기술 방향 | Rust 기반 Agent + eBPF 이벤트 수집 + AI 위협 분석 + 중앙관리             |
| 주요 고객 | 금융, 공공, 제조, 클라우드, 데이터센터, SaaS 기업                          |

## 2. 제품 포지셔닝

| 구분     | 내용                                     |
| ------ | -------------------------------------- |
| 1차 포지션 | AI Anti-Ransomware for Linux           |
| 2차 포지션 | Linux EDR                              |
| 3차 확장  | AI XDR / Autonomous SOC                |
| 차별점    | Rust 기반 안정성, eBPF 실시간 감시, AI 분석, 자동 복구 |

## 3. 핵심 컴포넌트

| 컴포넌트           | 역할                                  |
| -------------- | ----------------------------------- |
| Argos Agent    | Linux 서버에 설치되는 Rust 기반 보안 에이전트      |
| Argos Sensor   | eBPF, fanotify, auditd 기반 이벤트 수집 모듈 |
| Argos Detect   | 랜섬웨어 및 이상 행위 탐지 엔진                  |
| Argos Brain    | AI 위협 분석·요약·예측 엔진                   |
| Argos Response | 프로세스 차단, 격리, 네트워크 제한, 자동 대응         |
| Argos Recovery | 파일 백업, 스냅샷, 자동 복구                   |
| Argos Central  | 중앙관리 서버                             |
| Argos Copilot  | 자연어 기반 보안 분석 인터페이스                  |

## 4. 핵심 기능 요건

| 기능       | 상세 요건                                             |
| -------- | ------------------------------------------------- |
| 파일 행위 감시 | 생성, 수정, 삭제, 이름 변경, 권한 변경, 대량 암호화 패턴 감시            |
| 프로세스 감시  | 의심 프로세스 실행, 부모·자식 프로세스 관계, 비정상 명령어 추적             |
| 네트워크 감시  | 외부 C2 통신 의심 행위, 비정상 outbound 연결 감지                |
| 권한 상승 탐지 | sudo, su, setuid, capability 변경, root 권한 획득 시도 감시 |
| 랜섬웨어 탐지  | 대량 파일 변경, 확장자 변경, 엔트로피 증가, 짧은 시간 내 반복 쓰기 탐지       |
| 이상 행위 탐지 | 서버별 평소 행위 기준과 다른 파일·프로세스·네트워크 패턴 탐지               |
| 자동 차단    | 위험 점수 기준 초과 시 프로세스 종료, 격리, 경로 잠금                  |
| 자동 복구    | 손상 파일을 백업본 또는 스냅샷으로 복구                            |
| 정책 관리    | 경로, 사용자, 프로세스, 서버 그룹별 정책 설정                       |
| 중앙관리     | 에이전트 등록, 정책 배포, 로그 수집, 위협 현황 대시보드 제공              |

## 5. AI 기능 요건

| AI 기능                      | 상세 요건                                     |
| -------------------------- | ----------------------------------------- |
| AI Threat Summary          | 탐지 이벤트를 사람이 이해하기 쉬운 보안 사고 요약으로 변환         |
| AI Risk Scoring            | 이벤트, 자산 중요도, 사용자 행위, 위협 인텔리전스를 반영한 위험도 산정 |
| AI Root Cause Analysis     | 공격 시작 지점, 원인 프로세스, 영향 파일, 관련 사용자 분석       |
| AI Attack Timeline         | 공격 흐름을 시간순으로 자동 재구성                       |
| AI Response Recommendation | 차단, 격리, 복구, 예외 등록 등 대응 방안 추천              |
| AI False Positive Review   | 오탐 가능성 판단 및 근거 제공                         |
| AI Query Copilot           | 자연어 질문으로 보안 로그와 사고를 분석                    |
| AI Report Generator        | 사고 보고서, 임원 보고서, 기술 분석 보고서 자동 생성           |
| AI Threat Prediction       | 현재 패턴 기반 랜섬웨어 확산 가능성 예측                   |
